# Customer-Segmentation-using-K-Means-AI-Project-
This project applies K-Means clustering to segment customers based on income and spending behavior, enabling data-driven business strategies.

This project demonstrates how AI can be used to drive business decisions through customer segmentation.

## 🚀 Project Overview

This project applies K-Means clustering to segment customers based on income and spending behavior, enabling data-driven business strategies.

## 📷 Visualization

![Customer Segments](dashboard.png)

## 🧠 Approach

* Selected key features: Annual Income & Spending Score
* Applied K-Means clustering (k=5)
* Visualized clusters using matplotlib

## 🔍 Key Insights

* High-income, high-spending customers identified as premium segment
* High-income, low-spending customers represent growth opportunity
* Low-income, high-spending customers show strong engagement but potential risk
* Low-value customers contribute minimal revenue

## 🛠 Tools Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## 🎯 Business Impact

Customer segmentation enables targeted marketing, better customer retention, and optimized revenue strategies.
